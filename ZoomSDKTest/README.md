## How to Run

1. **Install Dependencies** (First time only)
   Open a terminal in the `ZoomSDKTest/server` directory:
   ```bash
   cd server
   npm install
   ```

2. **Start the Server**
   ```bash
   node server.js
   ```

3. **Open the App**
   Open your browser and visit:
   [http://localhost:4000](http://localhost:4000)
